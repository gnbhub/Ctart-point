## 1. 배열 활용 문제 풀이

배열을 한 번 더 복습하는 차원에서 문제를 만들어서 해결해보기로 했습니다. 문제는 다음과 같습니다.

```
"abcde"라는 문자열에 대하여 오른쪽(R) 또는 왼쪽(L)을 입력받은 뒤 숫자를 입력받아
입력받은 방향으로 입력받은 숫자의 크기만큼 이동시킨 문자열을 출력하여라.

입력예시 R 2

출력예시 deabc
```
